### **4단계: 생성 결과 확인**

생성된 파일들은 `outputs` 폴더에 저장됩니다:

```
outputs/
├── 스마일치과_마케팅제안서_202603.html     ← 최종 제안서 (이메일 호환)
├── 스마일치과_마케팅제안서_202603.pdf      ← 최종 제안서 (PDF)
├── 스마일치과_데이터분석_202603.json       ← 상권/경쟁사 데이터
├── 스마일치과_전략분석_202603.json         ← 마케팅 전략 데이터
└── 스마일치과_프로세스로그_202603.txt      ← 처리 기록
```

**HTML 파일:** 클라이언트에게 이메일로 직접 전송 가능
**PDF 파일:** 인쇄 또는 아카이빙용
**JSON 파일:** 데이터 재처리/분석용

## 🎨 제안서 구성 (3장 고정)

### **1장: 지역/경쟁 분석**
- 📍 상권 맥락 (지역특성, 배후수요)
- ⚔️ 경쟁 환경 진단
- 🚩 고객사 현재 포지션 (AS-IS)
- 🔎 경쟁사 Top 3~5 분석

### **2장: 채널 분석 & 전략**
- 📱 온라인 채널 추천 (네이버/SNS/지역앱 등)
- 📢 오프라인 매체 추천 (버스/엘리베이터TV 등)
- 🧭 포지셔닝 변화 전략
- 📣 채널별 진단 및 개선안

### **3장: 브랜드방향성 & 실행**
- ✨ 브랜드 방향성 (TO-BE 포지션)
- 🎨 Tone & Manner (톤앤매너)
- 🗣️ 핵심 메시지 & 카피
- 📅 월간 콘텐츠 플랜 (4주차)
- 💰 채널별 예산 비중

## 🔐 법규 자동 준수

시스템이 자동으로 업종에 맞는 법규를 적용합니다:

| 업종 | 적용 법규 | 주요 제약사항 |
|------|---------|-----------|
| 병원/치과/피부과 | **의료법** | ❌ '최고', '완치', '보장' 금지 |
| 보험/투자/은행 | **금융법** | ❌ '수익 보장', '무조건' 금지 |
| 뷰티/스킨케어 | **화장품법** | ❌ '만능', '질병 치료' 금지 |
| 카페/음식점 | **식품위생법** | ❌ '건강 음식', '약효' 금지 |

**모든 표현은 자동으로 검증되며, 법규 위반 시 수정됩니다.**

## ❓ FAQ

### Q1: 어느 정도의 정보가 필요한가요?
**A:** 최소 5가지 필수 정보(고객사명, 주소, 업종, 서비스, 타겟)만으로 시작 가능합니다. 추가 정보는 품질을 높입니다.

### Q2: 생성 시간은 얼마나 걸리나요?
**A:** 평균 3~5분(Step별 병렬 처리). 4개 에이전트가 순차 실행되므로 정보 복잡도에 따라 달라집니다.

### Q3: 의료법 표현을 위반하면 어떻게 되나요?
**A:** 자동으로 감지되어 권장 표현으로 수정됩니다. 최종 제안서 검증 시 법규 준수 여부가 표시됩니다.

### Q4: 기존 제안서를 수정할 수 있나요?
**A:** JSON 파일(데이터분석/전략분석)을 수정 후 Step 4(Styling)만 다시 실행할 수 있습니다.

### Q5: 여러 고객사를 한번에 처리할 수 있나요?
**A:** CSV 파일로 여러 고객사를 일괄 입력하여 자동 처리 가능합니다.
