# 🎯 마케팅 제안서 자동화 시스템 - 메인 워크플로우

## 📝 입력 정보 수집 폼

사용자가 아래 정보를 입력하면 자동으로 4단계 프로세스가 시작됩니다.

### **필수 입력 정보**

```
┌─────────────────────────────────────────┐
│ 고객사명:  ____________________         │
│ 주소:      ____________________         │
│ 주요업종:  □병원  □카페  □미용  □기타 │
│ 핵심서비스: ____________________        │
│ 주요타겟:  ____________________         │
└─────────────────────────────────────────┘

┌─────────────────────────────────────────┐
│ 예산범위 (선택):  ____________________  │
│ 추가정보 (선택):  ____________________  │
│ 출력형식 선택:                          │
│  □ HTML (이메일 호환)                   │
│  □ PDF (인쇄 품질)                      │
│  □ BOTH (HTML + PDF)                   │
└─────────────────────────────────────────┘
```

## 🔄 자동화 실행 프로세스

### **Phase 1: 입력 데이터 검증**

```python
# 입력 데이터 검증
{
  "고객사명": string (필수),
  "주소": string (필수),
  "주요업종": string (필수),
  "핵심서비스": string (필수),
  "주요타겟": string (필수),
  "예산범위": string (선택),
  "추가정보": string (선택),
  "출력형식": ["html", "pdf", "both"] (필수)
}
```

**검증 항목:**
- [ ] 필수 정보 모두 입력 확인
- [ ] 업종 키워드 자동 감지 (의료/금융/화장품/식품)
- [ ] 법규 적용 자동 설정
- [ ] 주소 정상 형식 확인

### **Phase 2: Step 1 실행 - Data Research Agent**

**요청 프롬프트:**
```
업종: {주요업종}
서비스: {핵심서비스}
주소: {주소}
타겟: {주요타겟}

위 정보를 바탕으로:
1. 상권 분석 (지역특성, 배후수요, 주요시설)
2. 경쟁사 분석 Top 3~5 (거리, 키워드, 규모, 특이사항, 검색현황, 선정사유)
3. 타겟 맞춤 진단 (서비스-타겟 매칭)

JSON 형식으로 반환하기
```

**출력 저장:**
```
outputs/[고객사명]_데이터분석_[년월].json
```

## 🔐 법규 자동 적용 로직

### **업종 감지 → 법규 자동 설정**

```
IF 주요업종 CONTAINS ["병원", "의원", "클리닉", "치과", "한의원", "피부과", "성형외과"]
  → 의료법 적용
  → Prompt에 [의료법_체크리스트] 포함

IF 주요업종 CONTAINS ["보험", "투자", "은행", "자산관리"]
  → 금융법 적용
  → Prompt에 [금융법_체크리스트] 포함

IF 주요업종 CONTAINS ["뷰티", "미용", "스킨케어", "화장품"]
  → 화장품법 적용
  → Prompt에 [화장품법_체크리스트] 포함

IF 주요업종 CONTAINS ["카페", "레스토랑", "음식점", "베이커리"]
  → 식품위생법 적용
  → Prompt에 [식품위생법_체크리스트] 포함
```

## ✅ 검증 체크리스트

실행 후 최종 검증:

```
【Step 1 검증】
✅ 상권 분석 완료

✅ 키워드/규모/강점 모두 기재
✅ 타겟 매칭 분석 완료

【Step 2 검증】
✅ 온라인 채널 3~5개 추천
✅ 오프라인 매체 2~3개 추천
✅ 포지셔닝 분석 완료
✅ 진단 & 개선안 모두 기재

【Step 3 검증】
✅ 브랜드 비전/USP 정의
✅ Tone & Manner 설정
✅ 핵심 메시지 2~3개 작성
✅ 4주차 콘텐츠 플랜 완성
✅ 법규 표현 검증 (금지/권장 확인)

【Step 4 검증】
✅ 3장 구조 완성
✅ 테이블 0개 (리스트만 사용)
✅ 이모지 활용
✅ 모바일 렌더링 확인
✅ 이메일 호환성 확인
✅ 의료법/금융법/화장품법/식품법 준수 확인
✅ 중복 내용 제거

【최종 배포】
✅ HTML 파일 생성
✅ JSON 결과물 저장
✅ 프로세스 로그 작성
✅ 모든 파일 outputs 폴더에 저장
```
