# AI 맞춤형 마케팅 제안서 생성 시스템 (AI Marketing Proposal System) Plan Document

> Version: 1.2.0 | Updated: 2026-03-09 | Status: Draft

## 1. Executive Summary (개요)
고객사(병원/일반업종) 정보를 바탕으로 **정밀 상권 분석**과 **경쟁사 벤치마킹**을 수행하고, 이를 4개 에이전트 팀이 협업 분석하여 **이메일 호환형 고품질 마케팅 제안서**를 생성하는 시스템입니다.

## 2. Goals and Objectives (목표)
- 고객사명 기반 자동 주소 확인 및 **반경 30km 내 상위 경쟁사(Top 4)** 정밀 분석
- 네이버 플레이스 순위, 리뷰 등 실질적 **평판 데이터**를 분석에 반영
- **표(Table) 사용 배제**, 모바일 최적화 카드뉴스 스타일의 제안서 생성
- 의료법 등 관련 법규를 준수한 안전하고 전문적인 콘텐츠 제안

## 3. Scope (범위)
### In Scope (포함 범위)
- **정밀 리서치 로직**: 
    - 주소 자동 검색 및 상권(유동인구, 배후세대) 분석
    - 핵심 서비스(진료과목/메뉴) 및 평판(리뷰, 순위) 파악
    - 반경 30km 내 경쟁사 3~4곳 발굴 및 전략 분석
- **AI 에이전트 협업 분석**:
    1. **데이터 리서치**: 위치 기반 상권 및 경쟁 환경 정밀 진단
    2. **마케팅 컨설턴트**: 온라인/오프라인 채널 추천 및 채널별 진단
    3. **콘텐츠 기획**: 4주간의 콘텐츠 로드맵(Hook+Message+CTA) 설계
    4. **스타일링 전문가**: Tone & Manner 설정 및 시각적 레이아웃 가이드
- **제안서 생성**: 6단계 구조(상권분석, 시장조사, 추천전략, 채널진단, 브랜딩, 예산/플랜)의 문서 출력

### Out of Scope (제외 범위)
- 실제 이메일 전송 API 연동
- 유료 상권 분석 툴 유료 데이터 구매 (공개된 검색/리뷰 데이터 활용)

## 4. Constraint & Success Criteria (제약 및 성공 기준)
| Category | Requirement |
|----------|-------------|
| **제안서 포맷** | **절대 표(Table) 사용 금지**. 리스트, 인용구, 이모지 활용 웹 브로슈어 스타일 |
| **의료법 준수** | 최상급 표현(최고, 유일 등) 사용 금지, 법적 가이드라인 준수 |
| **콘텐츠 구조** | 제목(Hook) + 핵심 메시지 + CTA 구조 필수 적용 |
| **리서치 범위** | 고객사 반경 30km 내 동종 업계 마케팅 상위 업체 포함 |

## 5. Proposal Structure (제안서 6단계 구조)
1. **정밀 상권 및 포지션 분석**: 지리적 특징, 배후 수요, 경쟁 환경 진단
2. **동종업 시장조사 (Top 4)**: 벤치마킹 포인트 및 선정 사유 명시
3. **브랜딩 및 마케팅 추천 전략**: 온/오프라인 최적 매체 조합 제안
4. **마케팅 채널 진단 및 개선**: 플레이스, 블로그, SNS별 실행 방안
5. **브랜드 방향성 (Tone & Manner)**: AS-IS -> TO-BE 변화 및 핵심 카피
6. **실행 예산안 및 4주간 콘텐츠 로드맵**: 주차별 테마 및 채널별 콘텐츠 설계

## 6. Agent Collaboration Strategy (에이전트 협업 전략)
1. **데이터 리서치**: 상권/경쟁사/평판 데이터를 수집하여 분석의 근거 마련
2. **마케팅 컨설턴트**: 수집된 데이터를 바탕으로 채널 진단 및 전략 방향성 수립
3. **콘텐츠 기획**: 전략 맥락에 맞는 4주간의 구체적인 메시지 가공
4. **스타일링 전문가**: 최종 결과물을 표 없이 시각적으로 풍부하게 구성 및 톤 조절
